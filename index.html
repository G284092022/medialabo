各授業回の学習事項

第1回 WWW と HTML

学習内容
Webページの作成方法、WWWについて学んだ。具体的には、リンクの構成やHTML文書の書き方、要素やハイパーリンクの追加などを実践しながら体験した。
演習の成果:演習1-4:HTMLのさまざまな要素

第2回 SCC による Web ページの装飾

学習内容
SCCを用いて、前回作成したこのWebページの装飾をした。グラデーションを自分の想定通りに装飾するのは難しかった。リンクにオーバーマウスすると色が変わる装飾はとても気に入っている。
演習の成果:演習2-3:ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

学習内容
JavaScriptを用いて,コンソールへの表示やオブジェクトについて学んだ。1~3回までの学んだことを活かし、数当てゲームやプロジェクト課題の編集を行なった。開発者ツールでエラーの確認や、コマンドでのデバックができるのも知った。いきなりVScodeで書くとエラーのときに原因がわからない時があるので、その前にコンソールに書いてみてから実行するのが大事だと知った。
演習の成果:演習3-6:JavaScriptのオブジェクト
